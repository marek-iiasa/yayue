1) Excel file 'dat1.xlsx' contains all input data for testing 'matouqin' model.

2) File 'dat1.dat' is generated by params.py. It contains parameters used in 'matouqin' main model.

3) File 'all_params' is generated by matouqin.py. It contains all parameters: input from excel file,
 parameters in data processing, parameters generated from params.py.

4) File 'data_structure' contains the parameter composition of the whole model.
